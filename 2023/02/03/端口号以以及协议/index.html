端口号以以及协议

TCP 代表传输控制协议

UDP 代表用户数据报协议

21 FTP服务 FTP服务器以及客户端所开放的端口，用于上传，下载。进行FTP文件传输中，客户端首先连接到FTP服务器的21端口，进行用户的认证，认证成功后，要传输文件时，服务器会开一个端口为20来进行传输数据文件

22 SSH端口 网络通信协议，通过命令行模式远程连接linux

23 Telnet端口 远程登录

25 SMTP 最初的SMTP端口

53 DNS 域名服务器

67，68 DHCP（动态主机分配协议） 为TCP/IP网络上动态配置的主机提供网络地址，服务器 用端口67，客户端用端口68

80 HTTP

110 POP3（邮局协议版本3） 用于将邮件从远程服务器检索到本地电子邮件客户端

143 IMAP（Internet消息访问协议） 用于本地客户端访问远程web服务器上的电子邮件

161 SNMP（简单网络管理协议） 是一组用于网络管理和监控的协议

443 HTTPS 安全超文本传输协议

587 SMTP 标准的安全SMTP（简单邮件传输协议）端口，默认的邮件提交端口

3389 远程连接服务器

8080 www代理服务

1433 SQL

3306 MySQL

1521 Oracle
